# Asterisk GUI for Railways

A web-based graphical user interface for managing and monitoring an Asterisk PBX system.

## 📌 Overview

This project was developed as part of my internship at **Western Railways** for learning and demonstration purposes.

Asterisk is a powerful open-source PBX (Private Branch Exchange) platform that is traditionally managed through a Command Line Interface (CLI). While CLI provides complete control, it can be challenging for users who are not familiar with terminal-based administration.

This project aims to simplify Asterisk management by providing an intuitive web-based GUI, allowing users to interact with the system through a browser interface instead of relying entirely on CLI commands.

The repository contains only the application source code and does **not include any confidential information, production data, credentials, or organization-specific configurations**.

---

## ✨ Features

- Web-based interface for Asterisk management
- User-friendly alternative to CLI-based operations
- System monitoring dashboard
- Interactive web pages for managing Asterisk-related tasks
- Backend integration with MySQL database
- Modular Flask-based backend architecture
